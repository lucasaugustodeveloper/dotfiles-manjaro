# @vinicius73/dotfiles - For archlinux

# Install
> need [yay](https://github.com/Jguer/yay) instaled

```shell
bash install.sh
bash arch-after-install.sh
```

```shell
bash arch-android.sh
bash arch-gnome-enviroment.sh
bash arch-packages-browsers.sh
```
